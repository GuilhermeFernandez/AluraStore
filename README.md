<h1 align="center"> Análise de Desempenho de Lojas </h1>
<p align="center"> Parte do Challenge Alura Store. O código conta com analise de diversas métricas das lojas contidas no Dataframe fornecido. Também conta com visualização em gráfico das diferentes métricas comparativamente. Por fim, também é possível avaliar o desempenho das lojas por um sistema de notas. </p>

<h2> Estrutura do Projeto </h2>
<p> O projeto é estruturado da seguinte maneira:<br>
        1. Leitura dos dados de cada uma das lojas;<br>
        2. Análises de todas as lojas para as seguintes informações: Faturamento, Vendas por Categoria, Avaliação das Lojas (Clientes), Maiores e Menores Vendas por Produto e Frete Médio;<br>
        3. Avaliação de Desempenho das Lojas com base nas informações anteriores, pontuando se a loja teve o melhor desempenho comparada com as outras;<br>
        4. Visualização dessas informações através de gráficos.<br>
        <br>
        Os resultados obtidos poderam ser visualizados em gráficos de pizza (pie), barra e linha, mostrando faturamento e sua relação com outros índices (frete, avaliação e maiores e menores vendas), bem como um gráfico para o desempenho.
</p>

<h2> Instruções para uso </h2>
<p> É recomendado que o arquivo enviado tenha as colunas com os nomes indicados, ou que o nome das colunas seja atualizado ao longo do texto.</p>
