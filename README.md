<h1 align="center"> Análise de Desempenho de Lojas </h1>
<p align="center"> Parte do Challenge Alura Store. O código conta com analise de diversas métricas das lojas contidas no Dataframe fornecido. Também conta com visualização em gráfico das diferentes métricas comparativamente. Por fim, também é possível avaliar o desempenho das lojas por um sistema de notas. </p>
